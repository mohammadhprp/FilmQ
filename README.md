## Getting Started
1.  Fork this repository to your GitHub account.

2. Clone the repo.
    ```sh
     git clone https://github.com/mohammadhprp/FilmQ/
    ```
3. Install packages.
    ```sh
     flutter pub get
    ```
4. Add The Movie Database API KEY on lib/model/api.dart
  ```dart
	 static const apiKey = "YOUR_API_KEY";
  ```

5. Run project.
    ```sh
     flutter run
    ```
